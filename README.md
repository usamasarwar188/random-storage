# Dropbox (Frontend w/ React.js)

![Dropbox Preview](./thumbnail.png)

This is the Dropbox CLone, a simples web app that allows users to upload files and save then in the cloud with Drag and Drop feature.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. Feel free to deploy if you want.

### Prerequisites

You will need the [backend](https://github.com/nubelsondev/dropbox-backend) to be running so that you can connect both sides of the Application.

```javascript
/*
Default Backend URL
$ http://localhost:3333
*/
```

### Installation

Clone the repository and run

```javascript
// With NPM
$ npm install
$ npm run start

// With Yarn
$ yarn install
$ yarn start
```
